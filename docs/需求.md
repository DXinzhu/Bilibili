界面描述：
最上方是页面标题 “推送设置”，标题下方紧接着是 “接收消息通知总开关” 选项（右侧标注 “[已关闭]”，还有箭头标识），下方配有修改路径的说明文字。
接着是 “消息免打扰” 选项（右侧有箭头），下方是对应的功能说明。
再往下是 “互动通知” 模块，包含三个选项：
“点赞”（右侧是开启状态的开关）
“评论”（右侧是开启状态的开关）
“@”（右侧是开启状态的开关）
之后是 “私信通知” 模块，只有 “聊天消息” 选项（右侧是开启状态的开关）。
接下来是 “关注通知” 模块，包含 “关注 up 主的更新提醒” 选项（右侧是开启状态的开关）。
然后是 “内容推荐” 模块，依次包含：
“推荐可能感兴趣的内容”（右侧是开启状态的开关）
“热点”（右侧是开启状态的开关）
“活动”（右侧是开启状态的开关）

阅读app的其他页面文件，仿照其文件拆分形式制作界面
