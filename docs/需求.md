这个页面是 “账号资料” 页面，布局是纵向的列表式样式：
最上方是 “头像” 区域，左边显示 “头像” 字样，右边是对应的头像图标；接着是 “昵称” 区域，左边是 “昵称” 字样，右边是当前昵称内容；再往下是 “性别” 区域，左边 “性别” 字样，右边是 “保密” 标识；之后是 “出生年月” 区域，左为 “出生年月”，右是对应的日期；接下来是 “个性签名” 区域，左是 “个性签名” 字样，右侧是可编辑的入口标识；再往下是 “学校” 区域，左 “学校” 字样，右是提示填写的文字；之后是 “UID” 区域，左 “UID” 字样，右是对应的 ID 数字；接着是 “二维码名片” 区域，左 “二维码名片”，右是二维码图标；再往下是 “购买邀请码” 区域，左 “购买邀请码”，右是剩余数量提示；最下面是 “哔哩哔哩认证” 区域，左 “哔哩哔哩认证” 字样，右侧是入口标识。
